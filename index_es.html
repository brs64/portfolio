Mis Proyectos

CARTOMAGIC

Descargar

Cartomagic es un juego de cartas mágicas compuesto por 13 cartas, generadas aleatoriamente en cada partida. El objetivo del juego es conseguir la mayor cantidad de puntos al final de la partida. Se juega en la línea de comandos (cmd) y fue diseñado para introducirnos al desarrollo en C++.

⌂ C++

VISOR DE DIAPOSITIVAS

Ver en GitHub

Esta es una aplicación que permite mostrar imágenes en forma de diapositivas. Una diapositiva se define como un grupo de imágenes con un título. Más tarde actualizamos la aplicación para que utilizara datos externos provenientes de una base de datos. El objetivo de este proyecto era enseñarnos cómo vincular SQL con otros lenguajes de programación y el desarrollo avanzado en C++.

⌂ C++

⌂ QT Creator

⌂ SQL
