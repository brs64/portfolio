BTSPlay - Media Asset Manager

Consulter

BTSPlay est un Media Asset Manager développé pour le BTS Audiovisuel de Biarritz. L'application permet de gérer, indexer et visualiser des vidéos stockées sur des serveurs NAS. Elle propose également le transcodage automatique entre formats MXF et MP4 via FFAStrans, un système d'authentification, et un panneau d'administration pour les professeurs. Développé en Laravel avec Docker, MySQL et FFmpeg.

Party Games

Consulter

Application web de jeux de soirée multijoueurs utilisant WebSocket pour une interaction en temps réel. L'ordinateur sert d'écran principal et les participants jouent avec leur smartphone. Inclut plusieurs jeux : "Qui serait le plus susceptible ?", générateur de fake news, jump coopératif, et capture de souvenirs de soirée. Stack technique : React, Node.js, Express, Socket.io, PostgreSQL et Redis, le tout dockerisé.

Batcher Docker

Consulter

Infrastructure complète développée lors de mon stage, utilisant Docker pour orchestrer plusieurs services. Le projet comprend Laravel pour le backend web, Node.js pour les services asynchrones, MongoDB pour le stockage de données non-relationnelles, et MySQL/MariaDB pour les données relationnelles. Architecture microservices avec PhpMyAdmin et Portkey pour l'administration.
